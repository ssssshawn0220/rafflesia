【段邢衍生】【一邢多段】太太

2020-01-27 19 min read # 肖鹏 # 周尚文 # 孟文禄 # 虞啸卿 # 段邢 # 龙虞

【泥塑/小妈/孕夫/流产/一邢多段】注意避雷

虞家小姐是在读书的时候得知的被许给孟家做填房太太的消息。落魄书生家能去这样的富人家，即便是填房，也算是极好的出路了。虞啸卿眼都不抬一下，白衫儿衬着红面颊，像极了一支复色二乔。虞父见他心不在焉，一把抢来他手上的书，翻了两页便气不打一处来：“西厢记？你上哪儿弄来的？”虞太太赶忙拉他回房，虞父嘴里还止不住嘟哝：“真是教坏了，早知道不该让他识字！还让他上街买东西！”

虞家小姐啸卿生得妙绝，纤细修长的身子上偏偏长了一张饱满的银盘面儿，唇凝脂香，再叫元宝领子一托，看上去便有了三分富贵。他偶尔为了家计出门，名气便在镇上传开，入了孟家老爷的耳朵。那边夫人刚走，孟家老爷就遣了仆人尚文上虞家聘他。

出嫁那天，虞啸卿在轿子里盯着自己褂上的牡丹团花，冠上垂下的珠儿随着轿子在眼前一滴滴地摇着，轿里红彤彤一片，虞啸卿觉得自己像是坐在了花心深处，心想起前些天读的“柳腰款摆，花心轻拆，露滴牡丹开”，一股暖流便漫到了脸上。

但他很快发现等待他的不是张生，红盖头下伸进来一只裹着松树皮一般的手——又老又会流油，剥了他的褂子在他皮肤上草草梭巡几下，便把自己那短小物什往他后边掼。他疼得要哭出来，但还没等他适应，那老头子便解决在他里面。他感觉后面像是被人吐了一口唾沫，可肇事者毫不知情地嘿嘿憨笑着，还把多余的几滴蹭到他腿根。眼泪蘸了胭脂洇湿了一片枕头，虞啸卿抽抽鼻子侧过脸，共枕的人已经鼾声隆隆了。

好在孟老头年近六十，这样难熬的夜晚并不多。虞啸卿因为识字的缘故，白天辅助老爷的事业，帮着清算账目，办事利索，颇得府上人喜欢。特别是准备继承家业的大少爷文禄，府内府外的事务一来二去，竟对太太生出了几分情愫。那天虞啸卿正在书房擎着毛笔看账本，大少爷趁四下无人抚上了他的手，虞啸卿惊叫一声，扰得一池的墨水斑驳满纸。下人们闻声进来，大少爷又装了一副正经模样：“花房新进了几种花，我不认识，能烦请姨母去看看吗？”下人们把脏污都收拾出去后，大少爷凑近他耳畔：“戌时花房，我等你。”大少爷厚实的嘴唇擦着他耳廓，弄得他有些痒痒的。虞啸卿隐约感觉不妙，二人年纪相仿，深夜相会花房，任怎么想也是些腌臜事。可自己日日这般和剃度出家又有几分差别？光阴易过催人老，辜负青春美少年。想起那件嫁衣，嫁来孟府后尚且有人好好打理，自己如花的年纪，竟比不得衣上的牡丹花叫人怜爱吗？

时至戌时，虞啸卿熄了自己房间的灯便往花房去，推开门来里面枝叶交错，黑压压地纠缠一片。他也不敢出声，往里悄悄走了两步，大少爷忽然从后边环住他，掰开高领热烘烘地吻上他的脖子。外边是老爷在唤各屋熄灯，虞啸卿被吓得赶忙要挣开，大少爷的五指陷进他的胸将他牢牢按在自己身前，压低声音道：“叫啊，你看爹是会休了你，还是会不要我？”虞啸卿便像那吹灭的红烛火焰一样，折了腰委了身。

大少爷的手坚硬有力，盖过他的嘴唇下巴，又划过他的脖子扯开他的领口，漂亮雪白的肌肤在月光下隐隐映出幽暗的光。大少爷把手伸了进去，冰得他一个哆嗦：“爹爹厉害，可他老了，你这么年轻，不寂寞吗？”

寂寞呀，这公事公办、干完就睡的性事说成是受刑也不过分，近些日子更是好久没找他了。但大少爷血气方刚，这双手像是要把他榨进自己身体。他这身软玉温香就是渴求这样的满怀一抱，可是他不敢说，憋得身上愈发灼热。

“你好烫啊……”大少爷一边挑逗他的耳孔一边将手更加深入，虞啸卿柔软的胸乳便溢了他满手。他捉住那点小核揉搓，虞啸卿的声音便像揉碎了的花瓣一样，艳艳地落下来。他难耐地勾了身后少爷的颈子求少爷的唇，少爷却作恶地偏开了头。强烈的热气喷在他赤裸的后颈，激起一片麻点，他不住地颤抖。少爷的手从旧式的裙袍里伸进去，他脑海里忽地出现了老爷掼进他身子的粗鲁，害怕得挣扎起来。挣扎中，白色的中衣被扯开一线，露出紧绷的一侧小腿和半幅鞋袜。他哀切地缩起身子，少爷却把他扳了过来，面朝自己摊开在浓绿的花荫下：“看清楚！现在干你的是谁！”

少爷的眼睛像狼，凝着月夜的漆黑与光泽。虞啸卿看不明白，恐惧、不解和委屈让他的唇瓣微微地颤着，少爷俯身吻他，动作意外地温柔：“我喜欢你，我会让你舒服的。”

于是虞啸卿像盛极的花朵躺在那里，少爷的手指轻轻碰他，衣服便一片一片地滑落。少爷脱下他的绣鞋，把他的双腿推到胸口，卸下全部防备的花心流着蜜水，让他滑溜溜地进去了。细密的喘息和细小的虫鸣，孟家的继承人和新来的继母，就这样在花房里缠缠绵绵了一夜。

晨钟之前虞啸卿回到了自己的房屋，仆人尚文随后跟了过来：“太太，要掌灯吗？”虞啸卿梳理着自己散乱的鬓发，看尚文这黑沉沉的脸许是一夜没睡，他应该是知道了。虞啸卿还没开口，尚文便说：“小的，什么都不知道……”

“我看你敢！”虞啸卿一个眼神杀向他，回归了平日行事果决的贤内助。尚文识趣，低头合上门出去了。

孟府度过了寻常的一晚。他们还是像往常一样办事，眼神交汇处，只有二人可见的擦枪走火。

后来虞啸卿和老爷做的时候，老爷问他衣服怎么补过，他假装云淡风轻地说是被枝条挂的。他看着老爷像一头蠢钝的老黄牛在自己身上蛮干，不禁想起了少爷热烈又温柔的爱抚。等老爷熟睡后，他只穿了件长衣偷偷敲开了大少爷的门，大少爷正诧异，他解开了前襟，鹅脂滑腻的肌肤便盈盈地亮在大少爷面前。大少爷赶忙把他打横抱进了屋子。

大少爷的爱抚如雨露滋润着太太的娇花，虽然只是短短几天不见，可是思念已经将娇花的土壤烤干。他一边抓着大少爷的手在自己身上摩挲，一边急切地嘤咛出声，两条腿缠着大少爷的腰把他拉扯到床上。

二人正翻云覆雨时偏房的二少爷肖鹏起夜，听到了些奇妙的声响，想着这新来的太太真是浪得很，父亲可是有福喽。解完手却忽然发现声音是从哥哥那里传来的。这下可了不得了！他蹑手蹑脚蹲到哥哥墙根下偷听起来。果不其然二人正芙蓉帐暖度春宵，肖鹏听得是脸红耳热，两腿之间也不听话地抬起了头，脑子里都是太太柔软的肉体骑在哥哥的身上，柳腰款款摆动，唇瓣轻启，小舌头像花信一样幽幽地勾出来。这个大哥，从小到大的诸事都是他占了便宜，这会儿连太太都捷足先登了。不行，非要给他点颜色看看。

约莫蹲了近一个时辰，天已经灰蒙蒙地亮起来。太太悄悄从大少爷房中溜出，被肖鹏一挡禁锢在墙角：“太太怎么从哥哥房里出来？”太太扯紧了衣襟，躲着肖鹏刀子一样的眼神。“别躲呀！”肖鹏邪笑着，脚伸到太太两腿之间，蹭了蹭流到太太小腿肚的液体，“这是什么？”太太脸上未褪的潮红又漫了上来。

“你给我过来。”肖鹏抓住他的手腕，把太太拽进房里按到床上把他衣服撕拉扯开，点点血色的吻痕昭示着方才战斗的激烈。太太委屈落泪：“衣服扯破，你...你不怕老爷发现吗？”肖鹏摸了一把他两腿间的粘液，亮晶晶的手指逼到他眼前：“你说，老爷会觉得是我撕碎的，还是哥哥撕碎的？”他只好咽着泪任肖鹏抠干净了大少爷方才洒进去的甘霖，又让他在自己身上肆虐了一番。

他回到自己房里，任头发散乱着出神。他觉得孟家像一个水磨，他就在里面被磨碎，被绞尽。可还有一个时辰就要开工了，他还要快快换上那重面具。他压低了声音抽泣，憋得嗓子疼。

这种时候尚文又进来了。他心里暗骂了一声不合时宜，速速收束起眼里的泪，干涸的痕迹在脸上纵横交错。尚文一言不发，停到他的面前将他拥入怀中。尚文进孟家之前做过苦力，胸腹上的肌肉坚实可靠。他窝在尚文怀里，让尚文的长衫裹住他的声音。

他想起前些日子听到小婢女们调笑说：“尚文长得像不像咱们大少爷？以后的大奶奶别认错了人哟。”他们两个着实长得像，就且当这是大少爷的怀抱吧。

那天以后肖鹏并没有放过虞啸卿，只要一有机会便强要他，可是虞啸卿在他的榻上像一条死鱼，从不主动索求，更不要说发出那一夜似的沉醉呻吟。肖鹏想他终究还是喜欢哥哥的。

虞啸卿以为日子就会这样一天天平安地过去，直到有一天早上他吐了，应该不是老爷的孩子，老爷两个月前外出做生意，但算了算信期也不知道是大少爷还是二少爷的。

太太告诉了大少爷自己怀孕的消息。大少爷脸上没有可能要当父亲的欣喜，他沉默了会儿问道：“这个孩子是我的，还是老二的？”

原来他知道！虞啸卿惊骇了。

“老头子的身体还有得几年好活，我呢，也不打算做李世民，老二是跟我不对付，可他真喜欢你，你对他说，这孩子是他的。”

虞啸卿恨恨地看了他一眼：“要是你的呢？”

“是我的，也不能是我的。”孟文禄咬着牙，忍着痛，他真馋太太的身子，也是真喜欢他，可他是孟家大少爷，后头有个弟弟虎视眈眈，他亲娘死的早，这些年二娘没少给老头子上眼药。

“要是让老头子知道，你和我都完蛋了。”他抱住自己年轻漂亮的后母，鼻息喷在他的后颈上，像他情热时的冲动。
“求你，这孩子要么是我弟弟，要么……是我侄子，否则，你就当没有他，好吗？”

虞啸卿能怎么办？他想留下这孩子，不为别的，这是他深宅大院唯一的依靠，孟文禄的话已将自己从他心中抹去，这个男人心里，他的前途才是第一。肖鹏……他想想就觉得恨意盈然，没有他，孟文禄未必会不认他。

这孩子已有两月，再不赶快，就没法让老爷以为是他的孩子了。

孟老爷还有三天就要从外地回来了，虞啸卿已做好了准备，老爷会欢喜他的，并且，那晚必定入港。

虞啸卿握紧了手中的小瓶，吩咐下人准备好大补的鳖汤。

还有一天，他按捺着紧张的心情。这些天他都不敢见人，生怕露出行迹，上次二太太已经有些意有所指地笑着喊他太太，说想给二少爷说个像太太这般年轻漂亮的媳妇。

他紧张，他恐惧，他惶恐，他憋闷，他觉得这深宅大院像一口古井，他，孟文禄，肖鹏，全是这古井中溺死的水鬼，而老爷，老爷是那古井里映着的幽幽月光，是他们唯一的指望。

而老爷回来的前一天，肖鹏又一次扯住了他，他眼中哀切，挣扎不得，被他拖进了书房。他护着自己的肚子，肖鹏反惊讶，太太今天怎么这般顺从？

不是顺从，他暗想，他想护住腹中一块肉，而不想让肖鹏知道。只是这宅院里各个精似鬼，谁也不知谁的底细，唯有他自己，风中之烛，袅袅弱弱，悬命如丝。

只是那日肖鹏发作起来似没有尽头，一下一下捅得他没完。他求：别，别，别要了，我受不住了。肖鹏仿佛没有听见，仍旧揽着他的腰，手掌压在那一块肉上。

“求你！二少爷！”他凄厉地尖叫，腹中仿佛连性命一起，被肖鹏的钩子勾去了魂魄。

“什么少爷，我他妈就是孟家的狗，什么好东西轮得到我？老东西回来了，你就准备做回你的太太了，孟文禄就打算做回他的好儿子，只有我，还是那个不成器的万年老二，对吧？我非得，我非得……” 肖鹏发了狂似的要他，听不见他的声音已经变了调了。

一股子血从他下身喷出来，沿着肖鹏的利器汩汩地流到了地上。

“这，这是怎么回事？！”惊叫的不止是二少爷，还有老爷。

——老爷提前回来了。

老爷通知了家人，到了地点却只有老大的亲随来迎接，老爷本身就有点疑惑，家中女眷该排成一排等待老爷，怎么今日如常？

大少爷的亲随欲言又止：“老爷，您自己看看去吧，这……实在不成话。”

老爷就呵退了一层层的护卫，八步并四，闯进了书房，就看见自己鲜花儿一样的新夫人，刚刚过门不到一年，就跟自己的二儿子绞在了一起，好看是真好看，年轻啊，漂亮啊，有力啊，他不是没看过活春宫，年轻的时候，也是喜欢买来雏儿看人开苞的口味，有时也自己亲自上阵给别人看，可没想到，老了老了，来了这么一出。二儿子的器官和鲜血一起流了出来，而新太太早已昏死了过去。

儿子是儿子，再烂也是儿子，可老子毕竟是老子，老子不给你，你不能抢，何况这腹中一块肉，明显不是短日子呢？

孟文禄悠悠叹了口气：“时候准吗？”
“准。”肖鹏午后必喝一碗红豆汤补气。

喜欢吗？孟文禄问自己，那日新娘进门，隔着祖宗牌位的一福，露出半截下颚，花儿般的嘴唇。自己有意勾引，那人却动了真情，深夜闯进自己的房门，连鞋袜都丢了，抱他进来的时候，不动心吗？

可新娘毕竟是新的娘，他们之间隔的不止是一个孟老爷，还有孟家数不尽的产业和祖宗。

虞啸卿在柴房里醒来，下身疼痛又不仅仅是疼痛，好像还有什么空虚的感觉。他摸上了肚子，知道是没指望了。

之后，肖鹏被发配到了孟家在边城的产业，十年内不许回来，那意思是由他自生自灭，二太太被老爷暴跳如雷地训斥了一顿，说她没管好儿子，关了一个月却轻轻松松地放了出来，也不知道用了什么狐媚手法，几个月后，新的姨太太进了门，这回老爷不让她碰账本了，本来嘛，伺候男人的身子，不该知书识字，心野了就会做出丑事来。

虞啸卿被关在柴房三天之后，血污和伤痕就沤得发臭，他察觉出下面是烂了。探指去够，血块粘在指上，黄的白的红的，他觉得恶心，可吐是没得吐了。

第四天，尚文撬开了柴房的窗户，给他带来了药和食水，他虚弱地探起头，尚文的眼泪滴在他的头发上。

怎么以前没发现，这宅子里唯一有人味的人是他呢？

尚文给他上药，咬着牙，抽着气，比他还疼的模样，然后又把饼掰碎了喂他，他吃不下，尚文就嚼碎了给他渡进去。

这么过了两天，第六天的时候，大少爷让亲随给了尚文柴房的钥匙，说：“以后走门。” 这大约是老爷默许的，毕竟虞啸卿的娘家还有人，死在这里，还是因为丑事，面上不好看。

于是一个月后，虞啸卿一瘸一拐地从柴房出来，开门的时候他看到了全宅的人。二少爷肖鹏是不在了的，二太太依旧风光无限，娇媚无限，看不出那一个月是何等咬牙切齿运筹帷幄，也不知孟文禄是用了什么手段，保下了虞啸卿的命。

虞啸卿收拾了小包袱从偏门离开的时候，吹吹打打的花轿刚刚抬到孟府门口。孟文禄没有来送他，但他叫来了尚文。

尚文的脸上有伤，右臂上有烙痕，但他永远不会对虞啸卿说，这伤是哪里来的。

“你跟着我干什么，”虞啸卿呵斥他。“你觉得我脏了，就轮到了你了？”

尚文依旧沉默，沉默到虞啸卿以为他哑巴了。

“不是的，太太。”尚文的声音很轻，很低，“太太，你原先读书的时候，我见过你。”

周尚文很久不说完整的话，说出来，很费劲，但他的眼神和手势足够比划出个大概，

那年三月初三，他和要好的男同学相约出游，龙姓男生穿着一套黑色的中山装，而他则一袭长袍，立在桃花树下，两人相视一笑。
“你等我，龙文章很认真地对他说，我要去重庆，跟我父亲见一见他的贵人，成功的话，就带你一起出去，过好日子。”
“那不成功呢？”
“我就回来，陪你在这小城里过一辈子。”
他信了。
等了又等，信是没收到的，他也不知道龙姓男生的父亲在政局里站错了队，而那人的结局，他蜗居小城，自然永远没了音讯。
龙姓男生终究是错过了，可那年在桃花树下路过的，不止一个龙文章。

周尚文那日刚刚送了母亲的灵柩上山，眼泪未干，昏沉中摔倒在山路上。

“那边有个人，咱们看看去，啊，龙文章，他还活着。”
“喂，你好了点没？你家在哪儿？”

周尚文想，真是前生冤孽，要是死在那场美梦里，也是他这辈子唯一一次大着胆子的绚烂。
可是活着，他就还是孟家的家生子，困在这里，这辈子不要想有个指望。
谁承想，那年的桃花飘落到了这枯井般的宅院里，又长在他近在咫尺的地方呢？

虞啸卿听完他断断续续的讲述，心头一片迷茫。

龙文章，这个名字，他当初是有多恨，恨到将两人来往的书信一寸寸撕碎，恨到将他碰过的衣服统统烧了个精光。

“我早就忘了，你也忘了吧。”他无力地摆摆手，走不动了。

周尚文转身就跑。虞啸卿笑了，果然，果然。

他撑着自己的腰，扶着桥，向下探着身子，水里的自己，离那年和龙文章的分别不过3年，离嫁进孟家不过十个月，一个月前，他还在大少爷的指尖盛放，而现在，鬓边已初现华发。

照花前后镜，花面交相映，低头弄莲子，莲子清如水，山寺月中寻桂子啊郡亭枕上看潮头，虞啸卿的脑海里没来由地想到了龙文章对他的念白，大少爷躺在他身畔时轻吻着他的鬓发，还有二少爷贪婪地吻他的膝弯。都是真爱过，都比不上别的。

他该往哪里去？往水里去吧。他只要松开手……

被一双有力的大手从桥上抢了回来，他跌落在周尚文强壮的怀抱里：
“太太，我去给您雇轿子了。”

桃花终究是要在有阳光和泥土的地方才能生长。
周尚文的臂弯没有织锦的绸缎和华丽的宝石，却有土有光。

自打太太被赶出孟家，尚文一守便是十年。他帮太太买菜、烧饭、打理家事，太太则做些文字活儿营生。太太脸上不再光洁的皮肤，新添的皱纹，在他看来都是共度时光的馈赠，他亲吻太太的唇，太多的人向那里索取青春和爱意，可如今这脉脉温软的心绪，他确信只有自己得此滋养。
